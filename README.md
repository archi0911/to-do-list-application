# 📝 To-Do List Web App

A simple and interactive **To-Do List application** built using **HTML, CSS, and JavaScript**.  
It allows users to add, update, complete, and delete tasks with data stored in **localStorage**.

---

## 📌 Features

- ➕ Add new tasks
- ✏️ Edit / update existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Persistent storage using `localStorage`
- 📋 Separate view for pending and completed tasks
- 🎯 Simple and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Font Awesome (icons)  
- Google Material Icons  
- Browser LocalStorage API  

---

## 🚀 How to Run the Project

1. Download or clone the repository  
2. Keep all files (`index.html`, `style.css`, `script.js`) in the same folder  
3. Open `index.html` in any web browser  
4. Start adding your tasks!

---

## ⚙️ How It Works

### ➕ Adding Tasks
- User enters a task in input box
- Task is saved in `localStorage` with a unique key

### 📋 Viewing Tasks
- All tasks are loaded from `localStorage`
- Completed tasks are filtered separately

### ✅ Completing Tasks
- Task moves from pending list to completed list
- Stored with prefix `completed-` in `localStorage`

### ✏️ Updating Tasks
- Click pencil icon to edit task
- Task is updated in `localStorage`

### 🗑️ Deleting Tasks
- Click delete icon to remove task permanently
